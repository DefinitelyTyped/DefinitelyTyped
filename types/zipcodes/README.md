# Installation
> `npm install --save @types/zipcodes`

# Summary
This package contains type definitions for zipcodes (https://github.com/davglass/zipcodes).

# Usage

```typescript
import * as zipcodes from 'zipcodes'

zipcodes.lookupByState('CA'); // returns a ZipCode[]
```
