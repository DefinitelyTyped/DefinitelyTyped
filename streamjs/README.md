# StreamJS Type Definitions

Unsupported StreamJS function / method signatures:
 * Stream(collection) 
 * Stream(string) 
 * Stream.empty() 
 * filter(sample) 
 * map(path) 
 * flatMap(path)
 * sorted(path) 
 * takeWhile(sample)
 * dropWhile(sample)
 * min(path) 
 * max(path) 
 * sum(path)
 * average(path)
 * allMatch(sample)
 * anyMatch(sample)
 * groupingBy(path)
 * toMap(path, mergeFunction)
 * partitioningBy(sample);
 * Optional.empty()
