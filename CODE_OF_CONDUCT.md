# Contributor Covenant Code of Conduct

This project follows the Contributor Covenant v2.0. We expect everyone participating in this project to abide by the code of conduct.

Please read the full text at: https://www.contributor-covenant.org/version/2/0/code_of_conduct/

If you experience or witness behavior that violates this code of conduct, please report it to the maintainers by opening an issue or contacting the repository maintainers on GitHub so we can respond promptly and appropriately.
